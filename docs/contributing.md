# Contributing Guide

Thank you for considering contributing to the Surface Pro 6 Linux Setup project!

## How to Contribute

### Reporting Issues
- Use the GitHub issue tracker
- Include your Surface Pro 6 model and specifications
- Provide kernel version: `uname -r`
- Include relevant logs and error messages

### Sharing Configurations
If you have optimized configurations:
1. Test thoroughly on your Surface Pro 6
2. Document the improvements
3. Submit a pull request with your config files

### Improving Documentation
- Fix typos or clarify instructions
- Add troubleshooting tips
- Translate documentation

### Code Contributions
1. Fork the repository
2. Create a feature branch
3. Test your changes
4. Submit a pull request with clear description

## Testing Needed
- Battery life measurements with different configurations
- IPTSD calibration values from different Surface Pro 6 units
- Compatibility testing on different distributions
- Hardware quirks and workarounds

## Community

- r/SurfaceLinux: https://reddit.com/r/SurfaceLinux

All contributions are appreciated!
